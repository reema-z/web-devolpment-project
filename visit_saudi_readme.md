# Visit Saudi 

## Overview

**Visit Saudi** is a web-based project designed to promote tourism in Saudi Arabia by showcasing popular destinations, cultural experiences, and travel information. The website provides users with an interactive and user-friendly platform to explore destinations while supporting basic user authentication features.

This project was developed as part of an academic course and focuses on applying web development concepts using **PHP**, **HTML**, **CSS**, and **MySQL**.

---

## Features

- Browse and explore Saudi tourist destinations
- User authentication system (login & logout)
- Reusable navigation bar using PHP includes
- Dynamic pages converted from HTML to PHP
- Database-driven user management
- Simple and clean UI design

---

## Technologies Used

- **Frontend:** HTML, CSS
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP (Apache)
- **Version Control:** Git & GitHub

---

## Setup Instructions

1. Install **XAMPP** and start Apache & MySQL
2. Place the project folder inside:
   ```
   C:/xampp/htdocs/
   ```
3. Import the database file into **phpMyAdmin**
4. Update database connection settings if needed
5. Open the browser and navigate to:
   ```
   http://localhost/visit_saudi
   ```
